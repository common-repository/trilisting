== Description ==

Trilisting plugin allows you to create beautiful directory listing websites.

[Demo](https://trilisting.com/places/) | [Documentation](https://trilisting.com/trilisting-plugin-documentation/) | [Free Theme](https://trilisting.com/#free-theme)

= Accounts & Profile​ =

Your customers can sign up, login and work from personal dashboard to manage their listings.

= Custom listing fields =

Would you like to add niche specific fields? Trilisting comes with full support of ACF plugin to manage any custom fields for your objects.

= Advanced search =

triListing provides powerful search capabilities. You may use simple search form or customize it by using any taxonomy types as filter.

= Reviews on board =

No need to use or even buy any additional plugins to let visitors leave feedback and rating. Trilisting has it on board already.

= Google ReCaptcha =

Stop bots and spammers by using powerful Google ReCaptcha. Included free of charge.

= Works with any theme =

triListing is ultimate directory listing plugin for WordPress and it works with any theme out of the box. Read documentation to learn more.

= Bookmarks =

Let your customers to save any listing to be checked later from their dashboard.

= WordPress widgets =

8 universal widgets to customize your directory listing website.

= Google Maps integration =

Full integration with Google Maps. Free bonus: Google Maps Marker Cluster included!

= Font Awesome =

Use 1300+ icons from the world’s most popular icon provider.

= Ajax listing loader =

Load additional items in runtime without page reload.

= Sticky listings =

Make any listing sticky (featured) and it will be always displayed on top of others.

= User functions =

Each user has access to the dashboard with following functions:

* Add listing
* Edit listing
* Delete listing
* Save listings (watch list)
* Favorite listings (wish list)
